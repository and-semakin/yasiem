git status – показать состояние дел в текущем проекте
git log – просмотреть лог изменений, отобразятся пояснительные сообщения в порядке хронологии.
git add <filename> – добавить файл к следующему коммиту
git commit -m "Пояснительное сообщение"
    or
git commit -v – в этом случае открывается vim, в первой строке нужно написать пояснительное сообщение к коммиту. Зато ниже можно будет просмотреть сделанные изменения, удобно.

git push origin master – отправить свои наработки на гитхаб
git pull – скачать чужие наработки с гитхаба
